tkArray

tkArray

A class containing static functions dealing with arrays. Do not use constructor.

Methods

(static) areElementsEqual(_arr) → {Boolean}

Source:
  • tamarack.js, line 170

Compare all the elements in array _arr with each other.

Example
tkArray.areElementsEqual(["blue", "blue", "blue"]);
// Output: true

tkArray.areElementsEqual(["blue", "blue", "red"]);
// Output: false

tkArray.areElementsEqual([false, false, 0]);
// Output: false
Parameters:
Name Type Description
_arr Array

The array whose elements are to be compared with each other
Returns:

True if all elements are equal (as compared by JavaScript's === comparer), false otherwise

Boolean
