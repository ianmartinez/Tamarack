tkColor

tkColor

A class containing various functions to convert between color types, grab colors from elements, generate random colors, parse color strings, and more. Used internally by tamarack to represent color.

(static) isColor(_color)

Source:
  • tamarack.js, line 338

Find whether _color represents a valid color that can be used. Currently, tamarack does not accept colors formatted in names, such as "black" or "transparent", instead use #000000 or rgba(0,0,0,0).

Example
tkColor.isColor("blah");
// Output: false
tkColor.isColor("#000000");
// Output: true
